Line & Angles

The Relationships You Need to Know

Linear Rules

Segment Addition Postulate

Formal Definition:
If points ABC are collinear, and B is between A and C, then AB + BC = AC

Explanation:
This Postulate is commonly used to prove that two segments on a line will add up to form a larger segment.

Angle Rules

Definition of Vertical Angles

Formal Definition:
Two angles whose sides form opposite rays

Explanation:
This is a setup definition to follow into the Vertical Angles Theorem. You use this rule to define if two angles are vertical angles, assuming you have been provided a diagram.




Vertical Angles Theorem

Formal Definition:
Vertical Angles are Congruent

Explanation:
This is a setup definition to follow into the Vertical Angles Theorem. It allows you to prove that two angles are congruent and is an extremely used set of theorems.

NOTE:
Make sure to use this definition and theorem in the correct order:

  1. Definition of Vertical Angles

  2. Vertical Angle Theorem
